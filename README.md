# Holy Grail Layout
Bare minimum for centered CSS Holy Grail layout with an optional max width.

The demo page has been tested and works flawlessly in:
  - Chrome 45
    - Windows
    - Android*
    - iPad*
  - Internet Explorer 10-11
    - Windows
    - Windows Phone 8.1*
  - Firefox 41
    - Windows
    - Android*
  - Safari iOS
    - iPad*
  - Safari 5.3
    - Windows
  - Stock Android browser v2.1*

_\* Small screen layout_
